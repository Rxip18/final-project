TYPOGRAPHY

This is an axial typrographic system, which means elements such as texts are portrayed on an axis or a line, just as I have done so here.
A radial typographic system is one that a central point with rays coming out of it creating a laser beam type effect.
A dilatational system is on where all elements expand from a point of focus to create a circular, or an arch style.
A random system typography with no rules or purpose when executing it.
The grid systems tends to work when there are line seperating the content, it could be horizontal or vertical or a square.
The modular typography system contains shapes, which give a bubble effect like the one shown here.
The transitional typographic system is one where the elements can move freely in a swivle type of way, these are typically used in posters and design web pages.
The bilateral typography system, is the most plain and boring system as, the content is usually set in the center of the screen, and does not really excite anyone when it's used.
